# Intro

Hey everyone!
I am working on a project investigating whether neuroeconomics can create RL algorithms that lead to less waste in organ allocation. I am basing my algorithm on the SCSRA algorithm proposed by Zhang et al. (Situational-Constrained Sequential Resources Allocation). 
This is an independent project I've been working on, on and off, since October (been focused on graduating from Caltech!). Lots of exploration to do and looking forward to updating with results. If you'd like to read the original research proposal, contact me by my email, ashlyn.roice@gmail.com. Otherwise stay tuned for updates!
